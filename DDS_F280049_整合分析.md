## 🔍 專案分析

### 280049_DDS 專案
**功能**: 高精度直接數位合成 (DDS) 波形產生器
- Q24 定點運算
- 狀態機管理 (4 種狀態)
- Slew Rate 控制
- 多波形支援 (正弦、梯形、DC)
- 120kHz ISR 更新

### F280049 專案  
**功能**: SPWM 控制系統 (已整合 ADC, CLA, Diagnostic)
- SPWM 三相控制
- ADC 電流/電壓採樣
- CLA 運算加速
- SPI 通訊
- 診斷功能

## 🏗️ 整合方案

### 方案 A: DDS 取代 SPWM (推薦) ⭐⭐⭐⭐⭐

**優勢**:
- ✅ DDS 功能更完整 (狀態機 + Slew Rate)
- ✅ 更高精度 (Q24 vs 浮點)
- ✅ 更好的參數控制

**整合方式**:
```
F280049/
├── DDS_Module/          [新增 - 取代 SPWM_Ctrl]
│   ├── DDS.c
│   ├── DDS.h
│   ├── DDS_UserConfig.h
│   ├── lut.c

├── CLA_Module/          [保留 - 可選用]
```

**優勢**:

**缺點**:

## ⚙️ 技術細節

### ISR 整合

**DDS 需求**: 120kHz ISR
**F280049 現有**: ePWM ISR

**整合方式**:
```c
__interrupt void INT_myEPWM0_ISR(void)
{
    // DDS 核心運算 (inline, 極快)
    uint16_t dacValue = DDS_Run_Core(&DDS);
    
    // 輸出到 DAC
    DAC_setShadowValue(myDAC0_BASE, dacValue);
    
    // 清除中斷
    EPWM_clearEventTriggerInterruptFlag(myEPWM0_BASE);
    Interrupt_clearACKGroup(INTERRUPT_ACK_GROUP3);
}
```

## ⚠️ 注意事項

### 1. DAC 輸出

DDS 輸出 12-bit DAC 值,需確認 F280049 有 DAC 模組。

**解決方案**:
- 使用 PWM 模擬 DAC
- 或使用外部 DAC

### 2. 記憶體需求

- **DDS 程式**: ~10 KB
- **查找表**: ~100 KB
- **總計**: ~110 KB

F280049 記憶體應足夠。

### 3. ISR 頻率

DDS 建議 120kHz,需確認 ePWM 設定。
